# quasar-project

Proyek latihan menggunakan Quasar
Dibuat oleh Jusuf Junior Athala

# how to use

1. install dengan 'npm init quasar'
2. pada directory, gunakan command 'quasar dev'
3. buka localhost http://localhost:8080/#/ pada internet browser
